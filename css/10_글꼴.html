# font family

  • 해당 요소의 글꼴을 정의하는 css 속성
  • 기본적으로는 해당 컴퓨터에 설치되지 않은 글꼴을 사용할 수 없다
  • 사용자의 컴퓨터에 글꼴이 존재하지 않는 경우에 대비하여 글꼴을 여러개 설정해 둘 수 있다
  • 사용자에게 필요한 글꼴을 설치하게 하는 것 보다 CDM이라는 개념을 통해 사용자가 글꼴을 받아서 허용하게 한다
  • CDM: Content Delivery Network, 다른 회사에서 인터넷에 올려놓고 배포하는 이미지, 글꼴 등의 리소스들을 의미한다

예시

글꼴을 확인하기 위한 문구, 글꼴을 제대로 확인하기 위해서 해당 언어의 자음과 모음을 모두 사용하여 만들어진 문구를 의미함 (A~Z, ㄱ~ㅎ)

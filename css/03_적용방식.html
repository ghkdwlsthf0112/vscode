# 웹 페이지에 css를 적용하는 방식 3가지

1.inline CSS (줄 내부CSS)
- stayle attr을 이용해 해당 요소에 스타일을 부여하는 방식
- 적용 우선 순위가 가장 높다
- stayle 속성의 값으로 CSS 문법을 넣을 수 있다
2. internal CSS (페이지 내부 CSSS)
- <style> 태그를 이용해 현재 웹 페이지에 스타일을 부여하는 방식
3. external CSS (페이지 외부 CSS)
- 외부의 .css파일을 불러와 사용하는 방식
-<link>태그를 사용해 외부의 css파일을 불러와 사용한다

# display 속성

  • 요소의 display 속성에 따라 해당 요소가 보이는 방식이 달라진다
  • div, p, h, ul, li, ...(기본 설정 block)
  • span, a, b, i, ... (기본 설정 inline)

# display block

  • 자리가 남더라도 할상 한 행을 모두 차지한다
  • 기본 너비는 100%로 설정되어 있다
  • 박스 모델을 사용한다
  • padding과 margin을 통한 여백 설정이 가능하다

# display: inline

  • 한 줄을 모두 차지하지 않고 필요한 만큼만 차지한다
  • 텍스트 사이에서 원하는 부분을 일부 선택할 때 주로 사용한다
  • padding과 margin이 제대로 적용되지 않는다

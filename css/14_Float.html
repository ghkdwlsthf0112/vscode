# Float

  • 해당 요소를 float으로 설정하면 다른 내용들이 해당 요소를 비켜가게 만든다
  • float: left로 설정하면 해당 요소가 왼쪽에 떠있게 되고 float: right로 설정하면 해당 요소가 오른족에 떠있게 된다
  • float을 같은 방향으로 설정한 요소는 차곡차곡 쌓인다
  • float을 다른 방향으로 설정한 요소는 각자의 방향으로 쌓인다
  • clear 속성을 통해 해당 방향에 요소가 계속 쌓이는 것을 막을 수 있다
  • float이 설정된 요소는 해당 요소를 포함한 부모 요소의 크기를 결정해 영향을 미치지 못한다

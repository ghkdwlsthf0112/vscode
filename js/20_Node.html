# Node

  • document 내부에 있는 모든 것을 Node라고 부른다
  • Element보다 더 작은 개념이다
  • 하나의 요소를 Element Node라거 부른다
  • 주석들은 Comment Node라고 부른다
  • 자바스크립트에서 새 노드 객체를 생성하여 기존 노드에 추가할 수 있다

# 자식 요소 추가하는 메서드

  • element.appendChild()
  • element.insertBefore(새 노드, 기준점 노드)
  • element.removeChild(node)
  • element.replaceChild(newNode, oldNode)
    방향키 왼쪽을 누르면 shift 방향키 오른쪽을 누르면 unshift 방향키 위쪽을 누르면 pushBtn 방향키 아래쪽을 누르면 pop이 동작하도록 만들어보세요

      버튼을 누르면 out에 별이 추가되거나 제거된다

# Document Object Model (DOM)

  • 웹 브라우저는 HTML코드르 해석하여 구조화된 객체 형태로 만들어 놓는다
  • 미리 만들어져있는 document라는 객첼르 통해 해당 웹 페이지의 HTML 요소에 접근할 수 있다

# documant에서 원하는 요소를 선택할 수 있는 메서드들

  • documant.getElementsById('id')
  • documant.getElementsByTagName('tagName')
  • documant.getElementsByClassName('className')
  • documant.getElementsByName('name')
  • documant.querySelector('cssSelector')
  • documant.querySelectorAll('cssSelector')

# 선택한 HTML 요소의 속성 및 메서드

  • getElement.innerHTML : 해당 요소의 내용에 접근
  • getElement.innerText : 해당 요소의 내용에 접근
  • getElement.속성이름 : 해당 요소의 해당 속성에 접근
  • getElement.setAttribute() : 해당 요소의 속성을 메서드로 변경
  • getElement.addEventListener : 해당 요소에 이벤트 추가
  • getElement.style.property : 해당 요소의 스타일에 접근
  • getElement.className : 해당 요소의 클래스 속성에 접근
